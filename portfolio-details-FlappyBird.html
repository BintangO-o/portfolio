Portfolio Details Flappy Bird

Deskripsi Proyek

Flappy Bird adalah salah satu karya portofolio saya dalam bidang web design. Proyek ini menampilkan detail rancangan dan implementasi game sederhana dengan tampilan interaktif.

Game ini dirancang untuk memberikan pengalaman bermain yang ringan, menyenangkan, namun tetap menantang. Dengan mekanisme sederhana — mengendalikan burung agar tidak menabrak pipa — proyek ini menjadi contoh nyata bagaimana konsep minimalis dapat menghasilkan daya tarik yang besar.

Tujuan & Fokus

Dalam pengerjaan proyek ini, saya menekankan beberapa aspek penting:

  • Estetika Visual: penggunaan warna cerah dan layout sederhana untuk menjaga fokus pemain.
  • Interaktivitas: memastikan kontrol game berjalan mulus dan responsif.
  • Optimasi Web: game dapat dimainkan dengan baik di berbagai perangkat, termasuk desktop dan mobile.
  • Pengalaman Pengguna (UX): menjaga agar pemain merasa tertantang namun tidak frustrasi, dengan tingkat kesulitan yang seimbang.
